# 📝 Note Pad

A simple and powerful note-taking application that runs directly in your browser.

🔗 **Live Demo:** [note-pad-aar6.onrender.com](https://note-pad-aar6.onrender.com/)

---

## ✨ Features

- 📌 Add and delete notes
- 🏷️ Create titles and assign category names
- 🔍 Filter notes by category in the search bar
- 🌗 Light and dark mode
- 👤 Create an account and log out
- 🎨 7 different color themes
- ↩️ Undo button
- 🔠 Font size adjuster
- 🖋️ 6 different font options
- **Bold text**, headings, numbered & bullet lists, horizontal dividers
- 📄 Export notes as plain text or PDF
- 📂 Upload documents
- 📋 Copy text within the editor
- 📖 Translates the written note into the desired language

## 🛠️ Technologies
- HTML5
- CSS3
- Vanilla JavaScript
